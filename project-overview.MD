# Project Overview

Create a plan for the following project and place the plan a markdown file in the codebase.

You are tasked with making a website for a company called Pristine Management. Pristine management is a HOA company that specializes in a certain type of HOA called a Metro District. The website really needs to point out the differences between a traditional HOA and a Metro District, what the differences are and how pristine management goes above and beyond to provide services to Metro districts. make sure to clarify that we do offer services for both but we specialize in Metro districts. below we have two different websites that we are using for the inspiration for this project. please draw upon inspiration for the copy of the website but please do not draw inspiration for the design of the website. Please use the front-end design skill when making design choices. Really lean into smart, sleek but minimal animations that really make the website pop but look professional. If you are going to be using inspiration of copy from the website please make sure to not take details such as the age of the business that would not be relevant to our business since this is a new business. Please do not make any copy on the page alerting to the fact that this is a new business just keep that in mind when creating copy for the website.
